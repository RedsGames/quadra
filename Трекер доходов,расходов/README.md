# Личный финансовый трекер (JSON Console Tracker)

Консольное приложение для учета личных финансов с хранением данных в формате JSON и возможностью экспорта отчетов.

## Особенности
- **Хранение данных:** Используется JSON-файл для обеспечения переносимости.
- **Экспорт:** Генерация текстовых отчетов в формате `.txt`, оптимизированных для Windows Блокнота (кодировка UTF-8-SIG).
- **UX:** Табличное форматирование в консоли и строгая валидация типов транзакций.
- **Логика:** Автоматический расчет баланса и нечувствительная к регистру фильтрация.

## Требования
- **Python 3.8+**
- Зависимости: `pytest` (для запуска тестов)

## Установка и запуск

1. **Клонируйте репозиторий** или скачайте файлы проекта.
2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Запустите приложение:**
   ```bash
   python main.py
   ```
   *При запуске программа предложит ввести имя файла базы данных (например, `finance.json`). Файл будет автоматически создан в папке `data/`.*

## Тестирование
Для проверки корректности работы модулей и чтения отчета о разработке выполните:
```bash
python tests/test_report.py
```

## Структура проекта
- `main.py` — Точка входа и инициализация.
- `data/` — Директория для хранения файлов баз данных (JSON).
- `src/` — Исходный код (Модели, Хранилище, Сервисы, Интерфейс).
- `tests/` — Исполняемые тесты и отчет о тестировании.
- `ЧЕКЛИСТ_Студента.md` — Отчет о выполнении требований ТЗ.
- `requirements.txt` — Список необходимых библиотек.
