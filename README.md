# Генератор QR-кодов
**Автор:** Спиричева Анастасия Сергеевна | **Группа:** ИС-31 | **Вариант №:** 21

Десктопное GUI-приложение на Python для генерации QR-кодов с возможностью добавления текстовой подписи и пакетной обработки ссылок из TXT-файла.

---

## Возможности

| Функция | Описание |
|---|---|
| Генерация QR | Создание QR-кода из любого текста или ссылки |
| Настройка размера | Регулировка box_size от 2 до 20 через Spinbox |
| Текстовая подпись | Добавление подписи под QR с автоматическим расширением холста |
| Выбор имени файла | Пользовательское имя файла, по умолчанию `qr_code.png` |
| Пакетная генерация | Загрузка TXT-файла со списком ссылок, создание QR для каждой строки |
| Сохранение в PNG | Сохранение через стандартный диалог файловой системы |

## Использование

### Одиночная генерация

1. В поле **«Ссылка или текст»** введите URL или произвольный текст.
2. В поле **«Размер QR»** выберите значение от 2 до 20 (по умолчанию 10).
3. В поле **«Текст под QR»** (необязательно) введите подпись — она будет добавлена под изображением.
4. В поле **«Имя файла»** укажите имя без расширения (если пусто — сохранится как `qr_code.png`).
5. Нажмите кнопку **«Создать QR-код»** и выберите папку сохранения в диалоге.

### Пакетная генерация из TXT

Подготовьте TXT-файл — по одной ссылке или строке на каждую строчку файла:

```
https://example.com
https://github.com
https://python.org
```

1. Нажмите кнопку **«Выбрать TXT файл»**.
2. Укажите путь к TXT-файлу.
3. Укажите папку, куда сохранить QR-коды.
4. Приложение создаст файлы `qr_1.png`, `qr_2.png`, … и покажет итоговый счётчик.

## Архитектура

```
QRGeneratorApp (tkinter.Tk)
        │
        ├── create_qr_image(data, caption, box_size)
        │       │  qrcode.QRCode  →  PIL.Image (QR-матрица)
        │       └── ImageDraw + ImageFont  →  добавление подписи
        │                (новый холст = высота QR + высота текста + 20px)
        │
        ├── generate_single_qr()
        │       считывает поля GUI → валидация → create_qr_image()
        │       → filedialog.asksaveasfilename() → img.save()
        │
        └── batch_generate()
                filedialog.askopenfilename() → читает TXT построчно
                for line: create_qr_image() → img.save(output_folder/qr_N.png)
                → messagebox с итоговым счётчиком
```

### Поток данных (одиночная генерация)

```
Поля GUI (data, caption, box_size, filename)
        ↓
generate_single_qr()  →  валидация входных данных
        ↓
create_qr_image(data, caption, box_size)
        ↓
qrcode.QRCode.make_image()  →  PIL.Image (QR)
        ↓ (если caption не пуст)
Image.new() + ImageDraw.text()  →  расширенное изображение с подписью
        ↓
filedialog.asksaveasfilename()  →  путь сохранения
        ↓
img.save(path)  →  PNG-файл на диске
```
